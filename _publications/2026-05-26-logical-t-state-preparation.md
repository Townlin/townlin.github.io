<div class="lang-block lang-zh" markdown="1">

本文比较三种有代表性的逻辑 T 态制备路线：魔法态蒸馏、魔法态培养和码切换。文章保留不同文献中的原始资源单位，记录输出错误率、单次尝试开销、每个接受输出的期望开销、占用空间、延迟和报告完整性，用于展示当前文献支持下的资源权衡。

</div>

<div class="lang-block lang-en" markdown="1">

This paper compares three representative logical T-state preparation routes: magic state distillation, magic state cultivation, and code switching. It keeps source-native resource units and records output error, single-attempt cost, expected cost per accepted output, footprint, latency, and reporting completeness to clarify the trade-offs supported by the current literature.

</div>
